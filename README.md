## react-tp-cli

![CI](https://github.com/Hitotsubashi/cicd-study/actions/workflows/ci.yml/badge.svg)

- react 项目脚手架 基于 webpack5
- 内置动态路由
- 可自由配置打包项
- 可选 Ts 模板

---

## 环境

全局安装 <a href="https://pnpm.io/"> pnpm </a>
pnpm 是个很好的包管理器

```
npm install -g pnpm
```

---

### 使用

- rt init
  - 请输入项目名称
  - 请选择模板类型
