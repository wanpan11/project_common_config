#### 内置功能

- typescript
- tailwind css
- eslint、prettier 代码自动检查、保存修复
- git 提交前自动修复
- 动态环境变量
- 动态打包配置
- 任务扩展
- 零黑盒
- 自定义常用表组组件

[TS 模版仓库](https://github.com/wanpan11/react-admin-tp)

#### 运行环境

全局安装 <a href="https://pnpm.io/"> pnpm </a>
pnpm 是个很好的包管理器

```
npm install -g pnpm
```

- rt init
